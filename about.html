About SKILLSCAN

SKILLSCAN is a mini project by Rohan Lokesh that aims to make resume screening practical and transparent.

What makes it different?

  • No inflated claims. This is a learning project.
  • Clear scoring based on TF‑IDF similarity + skill overlap with the JD.
  • Readable UI with explainable sections (skills, education, experience).

Limitations

  • Parsing quality depends on resume formatting and JD clarity.
  • Heavier semantic models are not included by default to keep hosting simple.

Roadmap: semantic matching (sentence-transformers), OCR for image PDFs, and HR portals with auth.
